Sequences

A sequence is an ordered list of items (numbers in maths). Each item in the list is referred to as a term.

Arithmetic Sequences

An arithmetic sequence is an ordered list of items, in which the numerical difference between the items is constant.

For example, an arithmetic sequence would be:

1,
5,
9,
13,
17, ...
+4
+4
+4
+4

Where the difference between each term is constant. In this case, the difference between each term is 4.

An example of a non-arithmetic sequence is:

1,
2,
4,
7,
16, ...
+1
+2
+3
+4

as the difference between each term is not constant.

Formulae

Finding a Term

An arithmetic sequence is generated by adding a constant to the previous term in the sequence. In our previous example, we started with the number 1 and added 4 to get 5. Then we added another 4 to get 9 etc.

To think about this in terms of a formula, we started with a number, which we shall call a1 and added the constant difference, which we shall call d, to it. Then the third term, we added the difference d again.

Our first term was a1
Our second term was a2 = a1 + d = a1 + (2 - 1)d
Our third term was a3 = a2 + d = a1 + d + d = a1 + 2d = a1 + (3 - 2)d

We can see a pattern emerging here. For an nth term, we add the difference d n - 1 times. Hence, we get our formula for the nth term:

an = a1 + (n - 1)d

Sum of an Arithmetic Sequence

If we add all of the terms of an arithmetic sequence up (until the nth term), we get

Sn = (a1) + (a1 + d) + (a1 + 2d) + (a1 + 3d) + ... + (a1 + (n - 1)d)

This can also be written in reverse, by starting from the nth term and taking away the difference, d.

Sn = (an) + (an - d) + (an - 2d) + (an - 3d) + ... + (an - (n - 1)d)

If we add these two equations together, we get:

Sn
=
a1 +
a1 + d +
a1 + 2d + ...
+ a1 + (n - 1)d
Sn
=

an - d +

a1 + d + an - d +

+ a1 + (n - 1)d + a1 - (n - 1)d
2Sn
=
a1 + an +
a1 + an +
a1 + an + ...
+ a1 + a1

This means that

2Sn = n[a1 + an]
⇒ Sn = n⁄2[a1 + an]
⇒ Sn = n⁄2[a1 + a1 + (n - 1)d]
⇒ Sn = n⁄2[2a1 + (n - 1)d]
